# pik

Color picker for terminal

# Installation

## From source
```
go install github.com/immanelg/pik
```

# Usage
## Flags
- `-hex string`: initial color

## Keybindings
- `h`, `l`, `H`, `L`: change current slider value
- `j`, `k`: change current slider
- `enter`: quit and print current color
- `q`: quit
